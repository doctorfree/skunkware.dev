2. The source

Looking at the source for this file will be instructive to show you how to use many of the Linuxdoc-SGML constructs. You should also read the Linuxdoc-SGML User's Guide, in the file guide.sgml.

The source looks and feels like LaTeX, as you can see. Paragraphs are separated by blank lines, macros are enclosed in angle brackets. It's quite simple.
