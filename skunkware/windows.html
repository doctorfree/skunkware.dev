Windows and UNIX Connectivity

Several Skunkware packages can be used to enhance a heterogeneous computing environment where Microsoft Windows and UNIX systems are maintained.

The Skunkware Windows and UNIX connectivity and integration packages include:

  • Samba - A Windows SMB/CIFS fileserver for UNIX
  • Windows CE to UNIX Connectivity Toolkit
  • WINE - Windows Emulation and Windows to UNIX porting library
