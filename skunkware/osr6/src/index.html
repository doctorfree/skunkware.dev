OpenServer 6 Skunkware Source Archives

23-August-2005

**NOTE:** This download area contains source archive stubs. No Skunkware source downloads are available here.

To download Skunkware source archives for OpenServer 6 visit ftp://ftp2.sco.com/pub/skunkware/osr6/src/

These are the sources used to compile the OpenServer 6 Skunkware packages. Pristine source archives can be found in the appropriate subdirectory. For instance, database managers can be found in the "db" subdirectory. Source code for packages built using RPM can be found in the source rpms in the "SRPMS" subdirectory.

Patches used to compile these on the OpenServer 6 platform can be found in the "patches" subdirectory.
