Installing Lxrun

  • “make install” copies the lxrun binary into /usr/local/bin, lxfront into /usr/local/linux/bin, the lxrun program interpreter ld-linux.so.1 into /lib and the HTML documentation into /usr/local/man/html/lxrun/

  • Linux shared libraries should be installed in /usr/local/linux/lib
