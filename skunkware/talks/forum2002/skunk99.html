Skunkware 99 Project Outline

SCO Skunkware 99 is intended to be an update CD-ROM containing Skunkware packages which have been added and/or revised since the release of Skunkware 98 or Skunkware 7.1 (OpenServer and UnixWare 7 respectively). The Skunkware 99 CD-ROM will be included in the SCO Forum 1999 attendees conference bags and available on request afterwards. It is not intended that Skunkware 99 replace either Skunkware 98 or Skunkware 7.1.

Note that i am using the term Skunkware rather than Open License Software Supplement. I intend to call this CD-ROM "SCO Skunkware 99". The term "Open License Software Supplement" refers to the CD-ROM which ships in the UnixWare 7 media kit.
