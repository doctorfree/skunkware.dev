Audio ReadMe

The /usr/skunk/audio directory hierarchy contains digitized audio in WAV, MPEG-2 and SND sound formats. Here's a brief description :

audio/wav :
Contains WAV format files suitable for playback with any WAV player. In particular, this directory includes the song "I Wanna Be Updated" by Deth Specula.
audio/snd :
Contains SND format files suitable for playback with play_snd.
audio/mp2 :
Contains MPEG-2 format files suitable for playback with maplay. In particular, this directory includes the song "I Wanna Be Updated" by Deth Specula.
