SCO Skunkware 98 Component Names

In order to install individual Skunkware 98 components using the Skunkware 98 CD-ROM, issue the command (as root): 

        # custom -p SKUNK98:COMPONENT_NAME -i -m /dev/rcd0
where COMPONENT_NAME is one of the component names given below and /dev/rcd0 is the device corresponding to the CD-ROM drive in which the Skunkware 98 CD is inserted.

To install individual Skunkware 98 components after downloading one of the tar archives of Skunkware media images, issue the command (as root): 

        # custom -p SKUNK98:COMPONENT_NAME -i -z /PATH_TO_MEDIA_IMAGES
where PATH_TO_MEDIA_IMAGES is the path to the directory in which the media images were extracted.
