mpeg_blocks

NAME
  mpeg_blocks - Analyze macroblocks in MPEG-1 bitstreams

SYNOPSIS
  mpeg_blocks mpegfile [ ordered | ordered2 | mbordered | fs4
  | fs2 | fs2fast | hybrid2 ]

DESCRIPTION
  mpeg_blocks provides an easy-to-use graphical interface to
  look at MPEG streams.  It operates by playing a video stream
  in one window, while another window displays the block type
  and motion vectors for each block in the current frame.

NOTES
  The player expects MPEG-1 video or system layer streams
  (without D-frames).

VERSION
  This is version 1.0, the initial public release.
