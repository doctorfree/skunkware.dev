IRC 2.8.21

IRC 2.8.21 provides an Internet Relay Chat client and server.

The installation and build notes are in the file doc/INSTALL.

The irc client users manual is in the file doc/Manual.

A description of the Note utility is in doc/NOTE.

The alt.irc FAQ is available as doc/alt-irc-faq.

The irc client manual page is doc/irc.html.

The ircd server daemon manual page is doc/ircd.html.

You may also be interested in Zircon 1.17p1, an X11 interface to Internet Relay Chat.
