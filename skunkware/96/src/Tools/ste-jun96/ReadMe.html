STE - Simple Text Editor

This is the June '96 version of STE from Robert Lipe

It incorporates several minor changes over the initial version.

  • Improvements to the help screen.
  • Turns off ISIG so ^Z doesn't get eaten by job control.
  • Addition of text search.

NOTE:
Reformat depends on an fmt binary that comes with the elvis [ a vi clone ] distribution. If you don't have that binary, reformat will not work. This functionality really should just be built into STE.
