Getting Started

Note: These notes pertain to the SCO Skunkware 96 CD-ROM
For information on ordering this free CD, see info.html. 

# mount -r -f RCKRDG /dev/cd0 /mnt
# /usr/bin/X11/netscape file:/mnt/index.html
If you do not have netscape installed, download a trial copy from http://athena.sco.com/drive/, install NCSA Mosaic off of this CD, or use any browser that supports tables and open the URL file:/mnt/index.html (assuming you mounted the CD on /mnt). If you do not have a graphical browser, you might want to check out Lynx 2.5FM a character browser available in custom installable form on this CD.

For further information on installation and use, see the file RELEASE.NOTES.

"Skunkworks" has long been used to denote underground special projects and engineering efforts not fully controlled by mainstream market demand. In keeping with this tradition, SCO is pleased to acknowledge the contributions of many technical professionals in the computing community in bringing you this collection of interesting software. These programs are provided for entertainment, education, and experimentation, possibly even real work.

Included are:

  • Custom installable media images for over 30 freeware and shareware packages
  • Many useful Tcl/Tk scripts along with Tcl 7.5, TclX 7.5.1 and Tk 4.1
  • Several useful internet-related tools such as Perl and freeWAIS-sf
  • Network/Security Related Tools
  • A snapshot of the SCO UnixWare FreeBird archives
  • Many useful X11 graphical clients including xanim, xpdf & xhtml
  • Images for backgrounds, instructions and examples
  • SCO Technical Level Supplements

Altogether, Skunkware 96 consists of approximately 340 Megabytes of useful, interesting or fun stuff for SCO UnixWare and approximately 300 Megabytes of useful and easy to install software for SCO OpenServer 5.

There are some useful tools here, but please use them with caution. SCO does not provide contract technical support for this software, and the software is not guaranteed to work right or be bug free. See other disclaimers below.

This cdrom is provided as a mountable filesystem, and may be mounted to view and use the contents. To mount the cdrom, use the mount(C) command with the mount point of your choice. For instance : 

      # mount -r -f RCKRDG /dev/cd0 /mnt
For further information on installation and use, see the RELEASE.NOTES file.
