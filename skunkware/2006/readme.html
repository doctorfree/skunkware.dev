Skunkware 2006 Edition
For SCO OpenServer 6

README

This README file covers how to install the skunkware programs that reside on this CD. Note: These applications are for OpenServer 6.0. They are in VOL format and should be installed using the custom program. Please follow the directions below to install each of the packages.

  1. Insert the CDROM into the CD drive.
  2. Mount the CDROM to the /mnt mount point:
    # mount /dev/cd0 /mnt
  3. From the command prompt type custom, or from within the GUI click on Software Manager.
  4. From the "Software"" pull down menu select "Install New."
  5. Select the default option of "From <your_server_name>."
  6. From the dropdown list of Media Devices, select "Media Image."
  7. Select "Continue."
  8. When prompted for the "Image Directory" type in the path to the package you want to install. Like, Image Directory:
    /mnt/vol_files/links-2.1pre22Sa
  9. Select Ok to continue.
  10. This should start the installation of the product. Follow the directions to complete the installation.

After the installation is complete for the package you selected to install, you may follow the steps to install another.
