skunkware/uw2/Packages

**NOTE:** This download area contains package stubs. No Skunkware downloads are available here.

These are pkgadd style packages of binary distributions for SCO UnixWare2. Download to a convenient temporary directory then install using pkgadd as in 

	pkgadd -d /download_dir_path/package_name

e.g for the file Perl.pkg downloaded to /home/hops/tmp 

	pkgadd -d /home/hops/tmp/Perl.pkg
