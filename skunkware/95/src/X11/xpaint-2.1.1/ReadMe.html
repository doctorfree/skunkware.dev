Description
A program to paint pictures and icons.
Productivity
This can create pictures displayable by the desktop.
Safety
Examined code, used it.
Work Planned
None. This has been linked against a modified libXpm.a; when XPM format files are produced, they can be displayed by the desktop in ODT 3.0.
Documentation
Manual pages and README file.
Verification
Fire it up and draw something.

How to get the source

To get a copy of the source sent to your Mosaic client in tar format click here. Note that all this source is configured to be installed under /usr/local/contrib. To build it for a different location, compile with make HCRDESTDIR=directory.
