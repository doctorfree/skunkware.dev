Description

Cpicker and pixedit make temporary changes to the installed colormap, allowing the user to observe the effects. It's useful for trying to pick that perfect color or find the appropriate color combination for an application.

Pixedit does it for a single pixel value.

Cpicker allows selecting from pixels in the colormap for a window.
