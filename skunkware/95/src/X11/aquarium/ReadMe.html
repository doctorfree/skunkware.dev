aquarium

     NAME
          aquarium - watch the fish

     SYNOPSIS
          aquarium [-option ...]

     DESCRIPTION
          Aquarium displays an odd assortment of object on your screen
          and moves them randomly about

     OPTIONS
          -fg foreground color

          -bg background color

          -display display
                  specify the display to use; see X(1).

          -count count
                  changes the number of objects from the default of 5.
