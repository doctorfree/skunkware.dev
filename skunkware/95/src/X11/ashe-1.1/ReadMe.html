Description

ASHE is an Editor for HTML Documents. It was written using C Language NCSA Mosaic Widget and Motif Library. The Visual Interface is based on NCSA Mosaic. ASHE has two main areas: the text area and the HTML browser area (WYSYWYG). You can edit your HTML document in the text area and display it in the HTML browse area.
Productivity
Useful for being able to quickly and easily edit HTML documents and be able to visually see the results immediately.

Build Notes

This source is configured to be installed under /usr/skunk. To install it in a different location, edit the Imakefile or simply copy the binary to the new location. It uses an app-defaults file which is installed in /usr/lib/X11/app-defaults.
