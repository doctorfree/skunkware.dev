ASHE README File

This is the ftp public ASHE Directory. ASHE stands for A Simple HTML Editor. This Editor was presented in the Second WWW Conference at Chicago . This is a beta version. We are not responsible for any damage to your files.

ASHE was written using C language, Motif and NCSA HTML Widget.

You can find the sources files in pub/puninj/ASHE/src . The name of the file is ashe.tar.Z . You can get the files following these steps: 


uncompress ashe.tar.Z
tar xvf ashe.tar
If you do not have the NCSA HTML Widget library, you can get it in pub/puninj/ASHE/libhtmlw.tar or in ftp.ncsa.uiuc.edu

The binary file xhtml (SUN Workstation only) is in pub/puninj/ASHE/bin. This was compiled for SunOS 4.1.3

More information about ASHE is in: http://www.cs.rpi.edu/~puninj/TALK/head.html
