UNIX system access commands (TCLX)

These commands provide access to many basic UNIX system facilities, including process handling, date and time processing, signal handling, linking and unlinking files, setting file, process, and user attributes, and the executing commands via the shell.

See the manual pages for the following commands:

  • alarm(TCLX)

  • chgrp(TCLX)

  • chmod(TCLX)

  • chown(TCLX)

  • convertclock(TCLX)

  • execl(TCLX)

  • fmtclock(TCLX)

  • chroot(TCLX)

  • fork(TCLX)

  • getclock(TCLX)

  • id(TCLX)

  • kill(TCLX)

  • link(TCLX)

  • mkdir(TCLX)

  • nice(TCLX)

  • readdir(TCLX)

  • rmdir(TCLX)

  • signal(TCLX)

  • sleep(TCLX)

  • system(TCLX)

  • sync(TCLX)

  • times(TCLX)

  • umask(TCLX)

  • unlink(TCLX)

  • wait(TCLX)
