About this book

 This SCO Visual Tcl™ Reference Guide contains information on using the SCO Visual Tcl scripting language to create OSF/Motif® graphical applications. Using this scripting language allows construction of sophisticated graphical applications without the complexity of directly using OSF/Motif or curses.

Tcl, the tool command language, is a simple generic scripting language designed to be embedded in applications and for easy extensibility. Extended Tcl, or TclX, is an extension to Tcl which adds commands for system access. SCO Visual Tcl makes use of Tcl and TclX, and extends them both with commands for generating graphical user interfaces (GUIs) and GUI components.

Tcl commands are described in manual page section (TCL), Extended Tcl commands in (TCLX), and SCO Visual Tcl commands in (VTCL). Complete command summaries for graphical and non-graphical commands are found in chpaters 4 and 5 of this guide.
