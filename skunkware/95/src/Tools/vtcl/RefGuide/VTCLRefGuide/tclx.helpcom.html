Help commands (TCLX)

The help facility allows one to look up help pages which were extracted from the standard Tcl manual pages and Tcl scripts during Tcl installation. Help files are structured as a multilevel tree of subjects and help pages. Help files are found by searching directories named help in the directories listed in the auto_path variable. All of the files in the list of help directories form a virtual root of the help tree. This method allows multiple applications to provide help trees without having the files reside in the same directory.

See the manual pages for the following commands:

  • apropos(TCLX)

  • help(TCLX)

  • helpcd(TCLX)

  • helppwd(TCLX)
