Chapter 5<P> Non-graphical commands Non-graphical commands

Non-graphical commands consist of standard commands (TCL), non-graphical SCO Visual Tcl commands (VTCL), and those found in the Extended Tcl command set (TCLX). This chapter consists of an alphabetical list of commands followed by lists grouped into logical areas for standard and extended commands.
