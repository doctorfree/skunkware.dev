How this book is organized

This book contains information about:

  • Chapter 1, ``Tcl - tool command language'', including descriptions of the language, command syntax, and variables and libraries used by Tcl.

  • Chapter 2, ``TclX - the Extended Tcl language'', describing the extensions provided by Extended Tcl 7.3b. This chapter also provides a complete list of TclX commands.

  • Chapter 3, ``SCO Visual Tcl - graphical scripting language'', describing program structures, callback data structures, resource files, system files, and class options.

  • all graphical commands included with SCO Visual Tcl.

  • all non-graphical commands included with SCO Visual Tcl.
Developers familiar with using Tcl and Extended Tcl may find it useful to skip ahead to Chapter 3, ``SCO Visual Tcl - graphical scripting language'', or directly to the command references.
