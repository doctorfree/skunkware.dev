TCP/IP server access (TCLX)

A command, server_open(TCLX), is provided to access TCP/IP based servers. In addition, it is easy to build servers using Extended Tcl that run under inetd. The fstat remotehost request is useful for these servers.

See the manual page for the following command:

  • server_info(TCLX)
