General commands (TCLX)

A set of general, useful Tcl commands includes a command to begin an interactive session with Tcl, a facility for tracing execution, and a looping command.

See the manual pages for the following commands:

  • commandloop(TCLX)

  • dirs(TCLX)

  • echo(TCLX)

  • infox(TCLX)

  • for_array_keys(TCLX)

  • for_recursive_glob(TCLX)

  • loop(TCLX)

  • popd(TCLX)

  • pushd(TCLX)

  • recursive_glob(TCLX)

  • showproc(TCLX)
