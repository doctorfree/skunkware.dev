Debugging and development commands (TCLX)

Extended Tcl contains the following commands and procedures that are useful for developing and debugging Tcl scripts. See the manual pages for the following commands:

  • cmdtrace(TCLX)

  • edprocs(TCLX)

  • profile(TCLX)

  • profrep(TCLX)

  • saveprocs(TCLX)
