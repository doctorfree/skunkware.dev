cequal(TCLX)
cequal -- compare two strings for equality

Syntax

cequal string1 string2

Description

This command compares two strings for equality. Returns 1 if string1 and string2 are identical and 0 if they are not. This command is a short-cut for string compare and avoids the problems with string expressions being treated unintentionally as numbers.
