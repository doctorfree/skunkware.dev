system(TCLX)
system -- execute command via system(S) call

Syntax

system command

Description

Executes command via the system(S) call. It differs from exec in that system does not return the executed command's standard output as the result string, and system goes through the UNIX shell to provide wildcard expansion, redirection, etc, as is normal from an sh(C) command line. The exit code of the command is returned.
