VxWidgetVarRef(VTCL)
VxWidgetVarRef -- return reference for per-widget frames variable

Syntax

VxWidgetVarRef widgetName varName

Description

Return a reference for the per-widget variable frames variable. This allows the variable to be passed by reference to other Tcl commands. For example: 
array names [VxWidgetVarRef my.widget.path data]

Options

widgetName
Name of widget that the variable is associated with.

varName
Name of the variable. May be a scalar or array, but not an element of the array.

Returns

A reference to the variable usable in the current scope.
