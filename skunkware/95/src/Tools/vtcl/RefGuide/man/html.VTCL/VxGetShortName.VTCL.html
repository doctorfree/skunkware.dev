VxGetShortName(VTCL)

VxGetShortName(VTCL)
VxGetShortName -- strip parent widgets off widget name, leaving short name

Syntax

VtGetWidgetShortName widgetName

Description

Given a widget name, strips off all the parent widgets, leaving the short widget name.

Options

widgetName
The widget name to strip.

Returns

The stripped widget name.
