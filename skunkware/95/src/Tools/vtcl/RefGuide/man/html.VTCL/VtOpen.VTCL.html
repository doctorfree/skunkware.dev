VtOpen(VTCL)
VtOpen -- establish connection to SCO Visual Tcl server

Syntax

VtOpen Class [options]

Description

Establishes a connection to SCO Visual Tcl. An optional second argument is the bookname for ScoHelp books. Returns the application name to be used as a parent for the top form.

Options

-helpBook bookname (C)
Specifies the bookname to reference for ScoHelp calls.
