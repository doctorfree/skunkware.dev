VxGetVar(VTCL)
VxGetVar -- get value of variable associated with widget

Syntax

VxGetVar widgetName varName

Description

Gets the value of the variable associated with the widget.

Options

widgetName
Name of the widget that the variable is associated with.

varName
Name of the variable. May be a scalar or array reference.
