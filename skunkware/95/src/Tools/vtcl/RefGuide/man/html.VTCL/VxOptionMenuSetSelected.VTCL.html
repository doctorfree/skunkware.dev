VxOptionMenuSetSelected(VTCL)
VxOptionMenuSetSelected -- set option menu's value to value referring to the label of the selected pushbutton

Syntax

VxOptionMenuSetSelected widgetName selection

Description

Sets the option menu's value to selection where selection refers to the label of the selected pushbutton.

Options

widgetName
Name of menu (as returned by VxOptionMenu).

selection
Label of option to be selected.
