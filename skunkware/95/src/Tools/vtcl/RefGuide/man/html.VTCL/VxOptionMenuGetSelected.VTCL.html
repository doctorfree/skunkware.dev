VxOptionMenuGetSelected(VTCL)

VxOptionMenuGetSelected(VTCL)
VxOptionMenuGetSelected -- return label of selected item in an option menu created by VxOptionMenu

Syntax

VxOptionMenuGetSelected widgetName

Description

Returns the label of the selected item in an option menu that was created by VxOptionMenu.

Options

widgetName
Name of menu returned by VxOptionMenu.
