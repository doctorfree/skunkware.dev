VtPullDown(VTCL)
VtPullDown -- create pulldown menu, return widget name

Syntax

VtPullDown object_name [options]

Description

Creates a Pulldown menu. (Label resources can be used to title the menu.)

Options

-cascadeButton (G)
This flag can be used only with VtGetValues. It returns the widget name of the pulldown's cascadeButton. Use this widget to set options such as -font.

-radioBehavior boolean (CS)
Specify that the Pulldown should act like a radiobox.

For example, to make a cascade menu within a pulldown: 

   ...
   set menubar [WsMenuBar $form.menubar]
   

   set pm [WsPulldown $menubar.phonemenu -label "Phone"]
   

   set check [WsPulldown $pm.check -label "Check"]
   	set check_yes [WsPushButton $check.yes -label "Yes"]
   	set check_no  [WsPushButton $check.no  -label "No"]
   ...
This creates a menu (Phone) containing the button check, which cascades to another menu that contains the buttons Yes and No.
