libwais

WAIS client libraries

Contributed by
Bob Gibson <rjg@sco.com>
Obtained from
The Internet via Ron Hess <hess@sco.com>
Restrictions
FreeWare from the Internet. Various copyrights strewn throughout.
Description
Libraries which provide direct WAIS support for applications (notably Mosaic and lynx)
Productivity
Without these libraries, Mosaic et. al. must use a WAIS gateway to access WAIS archives. These libraries allow WAIS servers to be contacted directly.
Safety
Widely used.
Work Planned
None.
Documentation
None.
Verification
Try opening a WAIS URL (e.g. "wais://cmns-moon.think.com/WWWP?") using a version of Mosaic built with these libraries.

How to get the source

To get a copy of the source sent to your Mosaic client in tar format click here. Note that all this source is configured to be installed under /usr/local/contrib. To build it for a different location, compile with make HCRDESTDIR=directory.
