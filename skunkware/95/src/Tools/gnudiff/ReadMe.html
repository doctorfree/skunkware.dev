Description
This is the GNU version of diff. Along with all the regular BSD diff features, this also has some new features. Look in the README file in this directory for a list of those new features.
Productivity
There are some nice features here. Especially the context differences and the ability to mark which function a diff appears in. It's supposed to be faster, too.
Safety
I trust GNU, don't you?
Work Planned
I'll try and stay up to new releases as they come out.
Verification
Use vanilla diff and GNU diff on the same pair of files. diff the results of each - they should match.

Note that all this source is configured to be installed under /usr/skunk. To build it for a different location, compile with make CTRBDESTDIR=directory.
