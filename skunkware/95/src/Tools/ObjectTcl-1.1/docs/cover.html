Object Tcl is a Tcl extension package that makes it possible to use object oriented programming concepts in Tcl and use/reuse C++ classes. Object Tcl acts like an interpreted extension to C++.

Object Tcl supports:

  • Classes and objects in Tcl
  • Multiple inheritance
  • Dynamic binding
  • Subclassing of unadulterated C++ classes in Tcl
  • C++ manipulation of Tcl objects that inherit from C++ classes
