Contributed software

We always welcome suggestions by our users, appreciate your patience and your work on improving GlimpseHTTP, adding to its functionality and bug fixes.

Here is a list of GlimpseHTTP variations and derived products known to developers so far:

  • "WAIS" version of aglimpse as (at least some time ago) used by Wired Magazine search site
  • GlimpseGN GlimpseHTTP for GN WWW/Gopher server
  • Standalone aglimpse
  • Tools for turning file tree into searchable, browseable HTML
    by Gary Capell (gary@cs.su.oz.au)
