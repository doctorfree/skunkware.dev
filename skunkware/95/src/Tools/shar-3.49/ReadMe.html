Pack and unpack files/directories for email/news

Contributed by

Obtained from

Restrictions

Description
Shar allows you to recursively pack up directories or files into "shar" format which is suitable for sending via email or news. Shar supports options for maximum file sizes uuencoding or compressing binary files, and loads of other options.
Productivity
This is a very handy way to pack up stuff for email. I used it all the time.

Examined code. Used it for years.
