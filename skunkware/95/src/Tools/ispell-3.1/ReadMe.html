ispell

Curses based Interactive spelling checker and correcter.

Contributed by
Larry Philps
Obtained from
uunet
Restrictions
GNU Copyleft
Description
ispell is a curses based interactive spelling corrector.
Productivity
Makes for very easy spell checking of documents. Elm 2.4 (coming soon) also understands ispell, and will allow you to spell check messages before they are send.
Safety
Used it for a few months
Work Planned
None
Documentation
Manual pages
Verification
Run ispell on some document with spelling errors.

Note that all this source is configured to be installed under /usr/skunk. To build it for a different location, compile with make CTRBDESTDIR=directory.
