SCO Skunkware Updates

The latest set of updates and fixes to SCO Skunkware 5 have been released on the SCO Skunkware 96 CD-ROM. This free CD targeted for both SCO OpenServer and SCO UnixWare systems and is available on request by e-mailing info@sco.com.

There are even updates to the updates at the SCO Skunkware 96 Updates site.

As additional components and fixes become available, we will make them available via ftp and WWW access. In order to view the contents of the SCO Skunkware Update tree, either :

ftp to ftp.sco.com
login as "ftp" using your e-mail address as the password and cd into the /Skunk directory. There you should find subdirectories ODT_3 and OpenServer_5. Change directory into the appropriate sub-directory. In each you should find directories CD-ROM and Updates. Change directory into the Update tree and issue the command : 
	get README /tmp/README.Skunk
You will then be able to view the file /tmp/README.Skunk on your local system. This file will detail the contents of the tree.
WWW access via www.sco.com
After having established an internet connection via a service provider, use a WWW browser like /usr/skunk/bin/X11/Mosaic and open the URL http://skunkware.dev/skunkware/95/osr5fix.html. This document will provide the details of downloading fixes and updates.

If you have bug reports, fixes, submissions, suggestions, compliments or complaints, feel free to email rr@ronrecord.com.
