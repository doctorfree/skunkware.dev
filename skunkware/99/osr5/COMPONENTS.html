UnixWare 7 Components Here.

In order to install individual Skunkware 99 components, issue the command (as root): 

On OpenServer 5
   # custom -p SKUNK99:default:COMPONENT_NAME -i -m /dev/rcd0
On UnixWare 7
   # pkgadd -d /dev/cdrom/c1b0t010 COMPONENT_NAME
where COMPONENT_NAME is one of the component names given below and /dev/cdrom/c1b0t0l0 or /dev/rcd0 is the device node name of your CD-ROM drive (this may vary from system to system).
