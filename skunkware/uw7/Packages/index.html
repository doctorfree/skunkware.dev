skunkware/uw7/Packages

These are pkgadd style packages of binary distributions for SCO UnixWare 7. Download to a convenient temporary directory then install using pkgadd as in 

	pkgadd -d /download_dir_path/package_name.pkg

e.g for the file nvi-1.79.pkg downloaded to /home/tmp 

	pkgadd -d /home/tmp/nvi-1.79.pkg
