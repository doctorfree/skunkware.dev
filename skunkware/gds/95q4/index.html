SCO Skunkware 5 GNU Development System Update

Robert Lipe (robertl@arnet.com) ported the GNU development system and tools to SCO OpenServer 5. Recently he has posted an update of the tools we included on SCO Skunkware 5. This update is available via anonymous ftp to ftp.sco.com in /Skunk/OpenServer_5/Updates/gds/scods.954b.tar (5.8 Mb).

The documentation and release notes for this distribution are also available via anonymous ftp to ftp.sco.com in /Skunk/OpenServer_5/Updates/gds/95q4-doc-html.tar.gz(HTML format, 0.9 Mb), /Skunk/OpenServer_5/Updates/gds/95q4-doc-ps.tar.gz(Postscript format, 1.5 Mb), and /Skunk/OpenServer_5/Updates/gds/95q4-relnotes.tar.gz(Release Notes).

You can read an overview that Robert provided describing what's included in this distribution and how to install it (using SCO custom !).

A document describing the patches Robert submitted is also available.

Finally, Robert provides us with a document describing how to build this release.
