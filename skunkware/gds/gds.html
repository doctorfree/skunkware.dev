SCO Skunkware GNU Development System

Robert Lipe (robertl@dgii.com) ported the GNU development system and tools to SCO OpenServer 5. Recently he has posted an update of the tools we included on SCO Skunkware 5 and SCO Skunkware 96. This update is available as a tar archive of custom installable media images in scods.954c.tar.

Kean Johnston (jkj@sco.com) has also ported the GNU development system and tools to both SCO OpenServer and SCO UnixWare. These ports are available as part of SCO Skunkware 97. You can find Kean's GNU dev sys at http://skunkware.dev/skunkware/devtools.

The documentation and release notes for Robert's distribution are available in

  • On-line HTML documents
  • 95q4-doc-html.tar.gz(HTML format)
  • 95q4-doc-ps.tar.gz(Postscript format)
  • 95q4-relnotes.tar.gz(Release Notes)

You can read an overview that Robert provided describing what's included in this distribution and how to install it.

You may want to read some brief answers to Frequently Asked Questions.
