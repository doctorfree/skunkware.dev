The Non-Stop Open Source E-Commerce Pack includes:

  • Apache 1.3.12 with PHP 3.0.14 and mod_ssl 2.6.0-1.3.12 (based on OpenSSL 0.9.4)
    Apache source and startup scripts modified to start on each node providing failover and scalabilit
  • The following have spawndaemon.d registration files and keepalive.d scripts as well as an init.d startup scripts that are NSC aware.
    • MySQL 3.22.30 built with support for UW7 kernel threads
    • PostgreSQL 6.5.3
    • eEMU 2.34, enterprise event management utility
    • Squid 2.3.STABLE2 web proxy cache
    • MiniVend 4.02, an Open Source e-commerce Internet application written in Perl
    • Webmin 0.78, web-based system administration interface
    • Enhydra 3.0b1, an Open Source Java and XML application server
  • MiniMate 4.0.0.1m, administrative interface for MiniVend
  • Perl 5.005_03 with Data-ShowTable 3.3, DBI 1.13, MySQL DBD 1.2210, and PostgreSQL DBD 0.93 modules
  • Glimpse 4.12.6, an index and search engine used by MiniVend
  • GNU Make, Bison, Flex
  • Graphics libraries (glibs)
  • GNU File utilities, GNU DBM, gzip/gunzip
  • GNU Shell utilities, Groff, wget
  • Java Servlet Development Kit, Swing (Java Foundation Classes)
