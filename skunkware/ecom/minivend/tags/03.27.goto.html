goto

CALL INFORMATION
Parameters: name if

Positional parameters in same order.

Pass attribute hash as last to subroutine: no

Must pass named parameter interpolate=1 to cause interpolation.

Invalidates cache: no

Called Routine:

ASP/perl tag calls: 

    $Tag->goto(
        {
         name => VALUE,
         if => VALUE,
        }
    )
  
 OR
 
    $Tag->goto($name, $if);

DESCRIPTION
NO DESCRIPTION
