SCO Skunkware 2000 Component Names

In order to install individual Skunkware 2000 components, issue the command (as root): 

        # custom -p SKUNK2000:COMPONENT_NAME -i -m /dev/rcd0
where COMPONENT_NAME is one of the component names given below and /dev/rcd0 is the device corresponding to the CD-ROM drive in which the Skunkware 2000 CD is inserted.
